# **App Name**: ScholarSlate

## Core Features:

- Offline Notebook: Allows users to create and manage handwritten notes using a stylus, fully functional offline.
- Document Reader: Enables reading documents with highlighting capabilities.
- Subject-wise Folders: Organizes notes and documents into subject-specific folders.
- Storage Manager: Manages local storage of notes and documents effectively. Provides file-system like controls for easy navigation.
- Note Converter: Tool to transform handwriting into text.
- Battery Efficient: Designed for long-lasting battery life, catering to extended usage without frequent charging.
- Budget Friendly: Cost-effective design to ensure affordability for college students.
- College Friendly: Strictly academic-focused to prevent misuse and ensure it is college-approved.
- Optimized for Reading and Writing: Specifically optimized for reading documents and handwritten note-taking using a stylus.

## Style Guidelines:

- Primary color: Dark navy blue (#2E3141) to provide a calming and focused environment, reminiscent of traditional notebooks.
- Background color: Off-white (#F0F4F8) for minimal distraction and better readability, offering a paper-like visual experience.
- Accent color: Soft sky blue (#77B5FE) to highlight important sections and interactive elements, drawing attention without overwhelming the user.
- Font pairing: 'Literata' (serif) for reading blocks of text in documents, and 'PT Sans' (sans-serif) for note titles and UI elements.
- Use clean, minimalist icons for folder navigation and tool selections.
- Simple and intuitive layout to enhance focus and minimize distractions.
- Subtle animations for folder transitions and page turns.
